## Important Tips for Generating:
1. **Aspect Ratio:** If you need wide background images, add `--ar 16:9` to the end of your Midjourney prompts. For square card images, leave it as is or use `--ar 1:1`.
2. **Icons vs Photos:** The website currently uses FontAwesome icons in the cards. You can replace the `<i class="fa-..."></i>` tags with `<img src="your-generated-image.jpg" class="card-img" />` and add some simple CSS to fit them nicely inside the cards.
